# 🔗 CIPHERLINK // Obfuscation Protocol

A futuristic **URL Shortener** built with **Flask + SQLite** backend and a **terminal‑style frontend**.  
Long URLs are converted into short codes (e.g. `ID-1A`) and displayed with a sleek design.

---

## 🚀 Features
- Convert long URLs into short codes instantly
- Copy shortened URL to clipboard
- Persistent storage using SQLite (`urls.db`)
- Clean terminal‑style UI with TailwindCSS
- Flask backend API for shortening and redirecting
